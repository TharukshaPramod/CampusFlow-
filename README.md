# CampusFlow

Smart campus hub with Spring Boot backend and React + TypeScript + Tailwind frontend.

## Quick start

### Backend
1. `cd backend/campusflow`
2. `mvn spring-boot:run`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Docker
- `docker-compose up --build`

## Structure
- backend: Spring Boot services and modules
- frontend: Vite React client
- .github: CI workflows
